To run neural network on MNIST dataset, you need to:
* [Download it](https://www.kaggle.com/datasets/aadeshkoirala/mnist-784)
* Put it in the datasets folder
* For 
  * Default neural network run [mnist.py](implemented_from_scratch/neural_network/mnist.py)
  * Convolutional neural network run [main.py](implemented_from_scratch/cnn/main.py)
